# Minivim – Uma IDE (Ambiente de Desenvolvimento): #

### Introdução: ###


Esse é **Minivim**, uma versão do Vim leve e avançada, perfeito para programas ou texto simples.

---

### Passo a passo: ###

1. Baixe o Termux no F-droid ou Github (obrigatório para usar o programa)

2. Baixe o git (caso não tenha instalado):
```
echo "Olá, Termux!"
ls -la
```

3. Clone o repositório assim:
 ```
git clone https://github.com/Tosso97999/Minivim/edit/main/README.md
 ```
4. Permitir permissão de execução:
   ```
   chmod +x main.sh
   ```
5. E pronto!, já tem um ambiente de desenvolvimento no seu Termux
